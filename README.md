# Netflix-Analysis--PowerBI
PowerBI Dashboard Analysing Netflix Dataset

Overview
This project presents a **Power BI Dashboard** analyzing Netflix data.  
The analysis provides insights into:
- Number of movies vs TV shows
- Genre distribution
- Content trends over the years
- Country-wise content availability
Files in this Repository
- `Netflix_Analysis.pbix` → Power BI dashboard file
- `README.md` → Project documentation

Dashboard Preview


How to Use
1. Download the `.pbix` file from this repo.
2. Open it in Power BI Desktop.
3. Connect to the dataset if required and explore the visuals.

Tools Used
- Power BI Desktop
- Netflix Dataset (Kaggle or other source)
